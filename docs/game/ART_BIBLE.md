# Art Bible

## Direction

Stylized premium pirate-investment hub: readable anime-inspired silhouettes, ocean-map lighting, metallic slab pedestals, and high-contrast analytics panels. The game should feel like a private captain's cabin and trading floor, not a generic marketing site.

## Environment

- Ocean plane with layered wave textures from Drive.
- Ship/island trading post with warm deck lights and teal/gold UI accents.
- Pedestals arranged like a card vault, with each slab angled toward the player.
- Bounty board for bids; archive room for research; theater corner for media.

## Characters

Use stylized private-use One Piece character assets where available. If high-quality character meshes are not available, first tranche can use color-coded proxy characters with portraits/UI labels while preserving the real game loop.

## UI

- Dark translucent panels, no nested card soup.
- Gold for positive/keep, red for cancel/avoid, teal for active/interactive, silver for neutral.
- Two-line dialogue summaries with deeper tabs for evidence.
- Source chips use source IDs from `data/research/source-ledger.json`.

## Asset Rules

- Every external asset must have provenance and private-use notes.
- Generated slab/pedestal GLBs are allowed.
- Meshy/SpriteCook can be used only after credit-cost confirmation.
