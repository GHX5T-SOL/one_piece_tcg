# Layout Inventory

Game layout rules:

- Full-bleed Unity canvas with minimal HUD overlays.
- Password-gated static shell before Unity load.
- Ship/island hub with landmark zoning: archive, bid board, card vault, media theater, market route, and risk clinic.
- Dialogue panel anchored bottom center with card preview side rail.
- Inventory/bid screens as game overlays, not separate marketing pages.
- PDF/report archive displayed in an in-game theater/archive menu and directly from `media/reports/`.

Archived video/report layout rules:

- Primary canvas is 16:9 landscape at `1920x1080`.
- Text-safe margins are `96px` left/right and `72px` top/bottom.
- Data panels use constrained widths between `560px` and `1280px`.
- Repeated data items may be cards; page/scene sections stay full-frame.
